# Legal Document Diff

Intelligent contract comparison tool for legal teams with semantic change detection.

## Features
- Clause-level diff detection
- Semantic change analysis
- Risk assessment per change
- Multi-format support (DOCX, PDF)
- Collaborative annotations

## Installation
```
pip install -r requirements.txt
```

## Usage
```
python main.py compare contract_v1.pdf contract_v2.pdf
python main.py risks contract.pdf --jurisdiction ID
```
